# note

## 网络模型和对应数据包的封装

| 网络层次 |   network layer   | 数据包 |  packet  |
| :------: | :---------------: | :----: | :------: |
|  应用层  | application layer |  报文  | message  |
|  传输层  |  transport layer  | 报文段 | segment  |
|  网络层  |   network layer   | 数据包 | datagram |
|  链路层  |    link layer     |   帧   |  frame   |
|  物理层  |  physical layer   |  比特  |   bit    |
